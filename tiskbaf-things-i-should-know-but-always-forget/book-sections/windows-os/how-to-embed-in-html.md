# How to embed in HTML

### Embed an image:

```html
<embed type="image/jpg" src="filename" width="" height="">
```

### Embed an HTML page:

```html
<embed type="text/html" src="URL" width="" height="">
```

### Using an iframe to embed:

```html
<iframe src="URL" width="" height=""></iframe>
```
