# Virtual Environments

### Creating a virtual environment

```bash
python3 -m venv [name]
```

{% hint style="info" %}
Remember to replace <mark style="color:yellow;">\[argument]</mark> with valid value.
{% endhint %}



### Activating the virtual environment

```bash
source [name]/bin/activate
```

{% hint style="info" %}
Remember to replace <mark style="color:yellow;">\[argument]</mark> with valid value.
{% endhint %}
